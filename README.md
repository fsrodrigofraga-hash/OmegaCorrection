# SAGE (Stochastic-background Analysis in Gravity Extensions)

SAGE is a python package for computing the stochastic gravitational-wave background (SGWB) generated by cosmological populations of compact binary coalescences within modified gravity frameworks.

The package is built upon and interfaces with several well-established community tools, including popstock, bilby, gwpopulation, and PyGWB, while providing a flexible framework for implementing modified gravity effects in SGWB predictions.

Additional examples, extensions, and the complete analysis pipeline will be released in this repository after publication of the accompanying paper.

Future updates will include support for additional examples of modified gravity theories, gravitational-wave detectors operating across different frequency bands, and a broader range of astrophysical source populations.
